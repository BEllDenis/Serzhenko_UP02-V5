Руководство пользователя информационной системы для управления научными конференциями
Добро пожаловать в руководство пользователя информационной системы для автоматизации работы научных конференций. Данное руководство поможет вам понять, как работать с системой в зависимости от вашей роли: заведующий подразделением, организатор или техник.
========================

1. Общие положения
Все пользователи системы должны пройти процесс авторизации, чтобы получить доступ к функционалу в соответствии с их ролью. После успешной авторизации вы сможете выполнять доступные вам задачи.
========================

2. Роль: Заведующий подразделением
Основной функционал:
Регистрация новых пользователей

Для добавления нового сотрудника в систему перейдите в раздел "Сотрудники".
Нажмите на кнопку "Добавить сотрудника", откроется окно для ввода данных нового пользователя (имя, логин, пароль, роль).
После заполнения всех полей нажмите "Сохранить". Новый пользователь будет добавлен в систему.
Изменение статуса пользователя

В разделе "Сотрудники" вы увидите список всех сотрудников.
Чтобы перевести сотрудника в статус "уволен", выберите нужного пользователя и нажмите "Изменить статус". Установите значение "уволен" и сохраните изменения.
После этого пользователь больше не сможет выполнять задачи в системе.
Формирование смен

Перейдите в раздел "Смены" для просмотра всех текущих смен.
Чтобы создать новую смену, нажмите "Создать смену". Укажите дату, время начала и окончания смены, а также назначьте сотрудников (организаторов и техников).
Сохраните смену для её активации.
Назначение сотрудников на смены

В интерфейсе смен выберите нужную смену.
Введите данные о сотрудниках (организатор, техник), которых хотите назначить на эту смену, и сохраните изменения.
Просмотр всех заказов

В разделе "Заказы" вы можете просмотреть все существующие заказы, их статус и текущий прогресс.
========================

3. Роль: Организатор
Основной функционал:
Создание нового заказа

Перейдите в раздел "Заказы".
Нажмите кнопку "Создать заказ". Откроется форма для ввода данных о конференции (название конференции, количество гостей, требуемое оборудование).
После ввода всех данных нажмите "Сохранить". Новый заказ будет зарегистрирован в системе.
Просмотр заказов за активную смену

В разделе "Заказы" вы сможете просмотреть все заказы, принятые за текущую смену.
Каждый заказ содержит информацию о конференции, статус выполнения и назначенных сотрудниках.
Изменение статуса заказа

Выберите нужный заказ в списке заказов.
Для изменения статуса заказа (например, "принят" или "не принят") нажмите "Изменить статус", выберите новый статус и сохраните изменения.
========================

4. Роль: Техник
Основной функционал:
Просмотр заказов

После авторизации перейдите в раздел "Заказы", где будет отображен список всех заказов, назначенных вам.
Выберите заказ для просмотра его деталей.
Изменение статуса заказа

Когда вы начинаете выполнять заказ, измените его статус на "готовится". Это можно сделать, нажав на заказ и выбрав "Изменить статус".
После завершения работы измените статус на "готов". Сохраните изменения.
========================

5. Авторизация и доступ
Процесс авторизации

Каждый пользователь должен ввести свои логин и пароль в форме авторизации на главной странице системы.
Если авторизация прошла успешно, пользователь получит доступ к функциям системы в зависимости от своей роли.
Сброс пароля

В случае, если вы забыли пароль, обратитесь к заведующему подразделением для его сброса или восстановления.
========================

6. Завершение работы
После завершения всех задач не забудьте выйти из системы, нажав на кнопку "Выход" в верхней панели управления.
========================

7. Техническая поддержка
Если у вас возникли проблемы при работе с системой, обратитесь в службу поддержки, предоставив описание проблемы и ваши учетные данные.
========================

Это руководство покрывает основные функции информационной системы для управления конференциями, позволяя пользователям эффективно выполнять свои задачи в зависимости от роли.
